1: Векторное произведение
Определение векторного произведения:
Векторным произведением векторов a и b называется вектор c, удовлетворяющий условиям:

    |c| = |a| · |b| · sin(a,b) - модуль равен произведению длин векторов на синус угла между ними
    c ⊥ a и c ⊥ b - перпендикулярен обоим векторам
    Если (a, b, c) - некомпланарны, то они образуют правую тройку

Обозначение: c = a × b
Свойства векторного произведения:
1. a × a = 0
Доказательство: a || a ⇒ a × a = 0 □
2. b × a = -(a × b)
Доказательство с чертежом:
Поскольку (b, a, b×a) - правая тройка, то b×a смотрит вниз перпендикулярно a и b
b × a = -(a × b) □
3. (a + b) × c = a × c + b × c (дистрибутивность)
4. λ ∈ R ⇒ λ(a × b) = (λa) × b = a × (λb) - ассоциативность
Доказательство:
Пусть a = (a₁, a₂, a₃), b = (b₁, b₂, b₃)
Тогда λa = (λa₁, λa₂, λa₃)
Вычислим (λa) × b


Продолжение доказательства свойства 4:
(λā) × b̄ вычисляется через определитель:

    Раскладывается по определителю с единичными векторами i, j, k
    Выносится λ за скобки
    Получается: λ(ā × b̄)

Аналогично для ā × (λb̄)
Итог: λ(ā × b̄) = (λā) × b̄ = ā × (λb̄) ✓
